## MOVIES — интерактивный каталог фильмов и сериалов

Проект — это мини-веб-приложение, стилизованное под современный стриминговый сервис.
Содержит динамический поиск, фильтры по категориям, страницы отдельных проектов и интерактивные карточки.

## Демо
Доступно на [GitHub Pages](https://hellogalina.github.io/movie-catalog-website/)

## Функциональность
- Главная страница с видео-заставкой
- Категории: Премьеры, Сериалы, Фильмы, Детям
- Поиск по базе фильмов (из movie.json) с подсветкой совпадений
- Горизонтальная карусель популярных фильмов с анимацией
- Адаптивное меню и бургер-меню
- Индивидуальные страницы для произведений:
 money-heist.html
john-wick.html
crime-and-punishment.html
the-boys.html
- Полноэкранные фоны и плавные hover-эффекты
- Используемые технологии: HTML, CSS, JavaScript

## Используемые библиотеки
- Bootstrap Icons
- Google Fonts (Roboto)

## Установка и запуск
Проект не требует сборщиков или серверов — работает как обычный статический сайт.

## Открыть в браузере
Просто открой index.html.
